The free open-source code of the MNDO program can be requested as described at https://www.kofo.mpg.de/en/institute/history/1993-to-present/theoretical-chemistry. 

To run calculations with the mndo program:

[path to mndo99 executable]/mndo99 < mndo.inp &> mndo.out
[path to mndo99 executable]/mndo99 < odm2.inp &> odm2.out
